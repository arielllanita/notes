
# Notes

## Demo

https://conote.netlify.app/

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/arielmeee/notes.git
```

Go to the project directory

```bash
  cd notes
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
